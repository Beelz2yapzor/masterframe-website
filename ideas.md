## Idea B: "Architectural Precision" — Swiss Grid Modernism

**Design Movement:** Swiss International Typographic Style + Contemporary Architecture

**Core Principles:**
1. Mathematical grid discipline — every element on a strict modular grid
2. Typography as architecture — type carries structural weight
3. Negative space as luxury — generous whitespace signals premium quality
4. Monochromatic depth with single bold accent

**Color Philosophy:**
- Background: pure white `#FFFFFF` with warm tint sections `#FAFAF8`
- Primary: deep charcoal `#1A1A2E` (architectural authority)
- Accent: forest green `#2D5016` (natural wood, Malaysian landscape)
- Secondary: warm sand `#D4C5A9` (timber, natural materials)
- Text: near-black `#111111`

**Layout Paradigm:**
- Strict 12-column grid with clear column gutters
- Hero: full-width with left-aligned text block, right 60% is full-bleed image
- Product catalog: clean 3-column grid with precise spacing
- Section headers: large oversized numbers (01, 02, 03) as decorative anchors

**Signature Elements:**
1. Thin horizontal rule lines as section separators
2. Oversized section numbers in light grey
3. Product specs displayed in clean table format with hairline borders

**Interaction Philosophy:**
- Minimal, purposeful — every interaction serves a function
- Filter system with instant visual feedback
- Smooth scroll with section indicators on right edge

**Animation:**
- Entrance: fade-up with subtle scale (1.02 → 1.0)
- Hover: underline grows from left on links
- Transitions: opacity-based, 200ms

**Typography System:**
- Display: Cormorant Garamond (elegant serif for headlines)
- Body: IBM Plex Sans (technical, precise, readable)
- Labels: IBM Plex Mono for measurements and specs

## Selected Design: **Idea B — "Architectural Precision"**

Swiss Grid Modernism with deep charcoal, forest green accent, and warm sand tones. Clean editorial typography using Cormorant Garamond + IBM Plex Sans. Strict grid discipline with generous whitespace signals premium quality. Oversized section numbers and hairline rules create architectural character.
